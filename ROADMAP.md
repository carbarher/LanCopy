# Roadmap

## Current priorities

- keep the safe default posture stable
- finish documentation for security and protocol behavior
- continue hardening tests around trust, TLS, and policy
- split large services into smaller policy/handler modules

## Staged direction

### Core

Safe LAN transfer with TLS, shared-folder confinement, resumable transfers,
and explicit trust controls.

### Advanced

Trusted-device permissions, sync, clipboard automation, diagnostics, and
other opt-in power features.

### Labs

Experimental protocol and architecture refactors that are not required for
normal file transfer.
