# Roadmap

## Current priorities

- keep the protected default posture simple and stable
- continue hardening tests around trust, protected paths, chat and remote operations
- improve first-run and reconnect flows for non-technical users
- split large services into smaller policy/handler modules
- keep public documentation aligned with the desktop app before every release

## Staged direction

### Core

Safe LAN transfer, disk/folder browsing, resumable transfers, protected system folders, and clear trust controls.

### Advanced

Trusted-device permissions, folder sync, diagnostics, bandwidth controls, remote power actions and other opt-in power features.

### Communication

A small, reliable chat between the two connected computers: multiline replies, sender labels, incoming-message focus and conversation history improvements.

### Labs

Experimental protocol and architecture refactors that are not required for normal file transfer.
