# FAQ

## Is LanCopy portable?
Yes. Main Windows download is a portable single-file EXE.

## Do I need internet?
No. Transfers are LAN peer-to-peer.

## Why installer if portable exists?
Installer is optional and can automate shortcuts/firewall setup.

## Why is a transfer retrying/failing?
Check receiver storage, firewall, antivirus, and compression suitability.

## CLI or API: which should I use?
- CLI: scripts and manual operations.
- API: integrations and automation from other tools/services.

## Where are integration examples?
- `README.md` integration sections.
- `scripts/api/*`
- `docs/wiki/CLI-API-Cookbook.md`
