Nuevo método GetStatusColor():

```csharp
private Color GetStatusColor(DownloadStatus status)
{
    return status switch
    {
        DownloadStatus.Pending => Color.LightGray,
        DownloadStatus.Downloading => Color.LightBlue,
        DownloadStatus.Completed => Color.LightGreen,
        DownloadStatus.Failed => Color.LightCoral,
        DownloadStatus.Cancelled => Color.Orange,
        DownloadStatus.Paused => Color.Yellow,
        _ => Color.White
    };
}
```
