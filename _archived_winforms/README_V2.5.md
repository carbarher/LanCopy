## 🚀 Cómo Usar

### **Dashboard**
```
1. Clic en "📊 Dashboard" en toolbar
2. Ver estadísticas en tiempo real
3. Identificar patrones de uso
```

### **Colecciones**
```
1. Clic en "📚 Colecciones"
2. Clic en "Nueva Colección"
3. Nombre: "Obras de Borges"
4. Tipo: Books
5. Agregar items manualmente
6. Sistema busca automáticamente
```

### **Calibre**
```
1. Descargar ebooks
2. Clic en "📖 Calibre"
3. Clic en "Sincronizar Descargas Recientes"
4. Libros agregados automáticamente
```

### **Notificaciones**
```
1. Minimizar ventana
2. Icono en bandeja del sistema
3. Notificaciones automáticas
4. Clic derecho para configurar
```
