═══════════════════════════════════════════════════════════════

FECHA: 15 Nov 2025 - 11:46 AM

OBJETIVO:
─────────────────────────────────────────────────────────────

Ver sinopsis, descripción y metadata de libros ANTES de descargarlos
para decidir si realmente te interesa el contenido.


CARACTERÍSTICAS:
─────────────────────────────────────────────────────────────

✅ Sinopsis Completa
   - Descripción del libro
   - Información del autor
   - Fecha de publicación
   - Número de páginas
   - Calificación promedio

✅ Múltiples Fuentes
   - Google Books API (primaria)
   - Open Library API (fallback)
   - Caché local de 24 horas

✅ Interfaz Elegante
   - Ventana dedicada con diseño moderno
   - Información organizada y legible
   - Botón directo para descargar

✅ Rendimiento Optimizado
   - Caché en memoria (MemoryCache)
   - Respuestas instantáneas para libros ya consultados
   - Timeout de 10 segundos por API


CÓMO USAR:
─────────────────────────────────────────────────────────────

1. Busca un libro en la pestaña "🔍 Buscar"
2. Haz clic derecho sobre un resultado
3. Selecciona "📖 Ver Sinopsis"
4. Espera unos segundos mientras se obtiene la información
5. Lee la sinopsis y decide si descargar

FUENTES DE DATOS:
─────────────────────────────────────────────────────────

1. GOOGLE BOOKS API:
   - URL: https://www.googleapis.com/books/v1/volumes
   - Datos: Título, autor, descripción, fecha, páginas, rating
   - Ventaja: Más completo, mejor calidad
   - Limitación: No todos los libros están disponibles

2. OPEN LIBRARY API:
   - URL: https://openlibrary.org/search.json
   - Datos: Título, autor, descripción, año, páginas
   - Ventaja: Biblioteca más amplia
   - Limitación: Menos metadata, sin ratings


EXTRACCIÓN DE TÍTULO Y AUTOR:
─────────────────────────────────────────────────────────

El sistema detecta automáticamente el título y autor del
nombre del archivo usando patrones comunes:

✅ "Autor - Título.epub"
✅ "Título - Autor.pdf"
✅ "Autor_Título.mobi"
✅ "Título (Autor).epub"
✅ "[Autor] Título.pdf"

CACHÉ DE METADATA:
─────────────────────────────────────────────────────────

El sistema guarda en caché las sinopsis consultadas:

- Duración: 24 horas
- Ubicación: Memoria (MemoryCache)
- Clave: "synopsis_{título}_{autor}"
- Beneficio: Respuesta instantánea en consultas repetidas
