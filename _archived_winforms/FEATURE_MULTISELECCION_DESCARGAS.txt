EJEMPLO DE CÓDIGO:
─────────────────────────────────────────────────────────────

ANTES (solo primer archivo):
  private void CancelDownload()
  {
      if (lvDownloads.SelectedItems.Count == 0)
          return;
      
      var fileName = lvDownloads.SelectedItems[0].Text;
      
      if (activeDownloads.ContainsKey(fileName))
      {
          activeDownloads[fileName].Cancel();
          lvDownloads.SelectedItems[0].SubItems[3].Text = "⏸️ Cancelando...";
      }
  }

AHORA (todos los archivos):
  private void CancelDownload()
  {
      if (lvDownloads.SelectedItems.Count == 0)
          return;
      
      int cancelledCount = 0;
      
      // Procesar TODOS los archivos seleccionados
      foreach (ListViewItem item in lvDownloads.SelectedItems)
      {
          var fileName = item.Text;
          
          if (activeDownloads.ContainsKey(fileName))
          {
              activeDownloads[fileName].Cancel();
              item.SubItems[3].Text = "⏸️ Cancelando...";
              cancelledCount++;
          }
      }
      
      if (cancelledCount > 0)
      {
          Log($"⏹️ {cancelledCount} descarga(s) cancelada(s)");
      }
  }


CASOS DE USO:
─────────────────────────────────────────────────────────────

1. CANCELAR MÚLTIPLES DESCARGAS:
   - Seleccionar 10 descargas lentas
   - Clic derecho → "❌ Cancelar descarga"
   - Todas se cancelan instantáneamente

2. CAMBIAR PRIORIDAD EN LOTE:
   - Seleccionar 5 archivos importantes
   - Clic derecho → "⭐ Prioridad" → "🔴 Urgente"
   - Todos pasan a prioridad urgente

3. PAUSAR TEMPORALMENTE:
   - Seleccionar 3 descargas grandes
   - Clic derecho → "⏸️ Pausar"
   - Todas se pausan para liberar ancho de banda

4. REINTENTAR FALLIDAS:
   - Filtrar por "Fallidos"
   - Seleccionar todas (Ctrl+A)
   - Clic derecho → "🔄 Reintentar descarga"
   - Todas se reintentan en paralelo

5. REANUDAR PAUSADAS:
   - Seleccionar descargas pausadas
   - Clic derecho → "▶️ Reanudar"
   - Todas vuelven a la cola

PROCESAMIENTO EN PARALELO:
─────────────────────────────────────────────────────────────

El método RetryDownload() procesa las descargas en PARALELO
usando Task.WhenAll para máxima eficiencia:

  var retryTasks = tasksToRetry.Select(async pair =>
  {
      try
      {
          await ProcessDownload(pair.task);
      }
      catch (Exception ex)
      {
          Log($"❌ Error al reintentar {pair.task.File.FileName}: {ex.Message}");
      }
  });
  
  await Task.WhenAll(retryTasks);
