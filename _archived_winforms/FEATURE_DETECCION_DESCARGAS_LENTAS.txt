PROBLEMA:
─────────────────────────────────────────────────────────────

Cuando una descarga es muy lenta (usuario con conexión lenta,
saturado, o problemas de red), el archivo puede tardar HORAS
en completarse, cuando podría haber otros usuarios con el mismo
archivo descargando mucho más rápido.

Escenarios problemáticos:
  ❌ Descarga a 5 KB/s (tardaría 20 minutos para 5 MB)
  ❌ Usuario no se da cuenta de la lentitud
  ❌ Otros usuarios tienen el archivo a 500 KB/s
  ❌ Tiempo desperdiciado esperando


SOLUCIÓN:
─────────────────────────────────────────────────────────────

✅ Monitoreo automático de velocidad de descarga
✅ Detección de descargas lentas (< 10 KB/s)
✅ Búsqueda automática de proveedores alternativos
✅ Cambio automático a usuario más rápido
✅ Logs informativos del proceso


PARÁMETROS CONFIGURABLES:
─────────────────────────────────────────────────────────────

1. MIN_SPEED_KBPS = 10.0
   - Velocidad mínima aceptable: 10 KB/s
   - Si está por debajo, se considera "lenta"

2. SLOW_DOWNLOAD_CHECK_SECONDS = 60
   - Verificar velocidad cada 60 segundos
   - Esperar 60s desde inicio antes de verificar

3. SLOW_DOWNLOAD_THRESHOLD_CHECKS = 3
   - 3 verificaciones consecutivas lentas
   - Después de 3 checks, buscar alternativa


FUNCIONAMIENTO:
─────────────────────────────────────────────────────────────

1. INICIO DE DESCARGA:
   - Se inicia descarga normalmente
   - Se registra tiempo de inicio
   - Se registra bytes descargados

2. MONITOREO CONTINUO (cada 60s):
   - Calcular bytes descargados desde último check
   - Calcular velocidad actual en KB/s
   - Comparar con MIN_SPEED_KBPS (10 KB/s)

3. DETECCIÓN DE LENTITUD:
   - Si velocidad < 10 KB/s → Incrementar contador
   - Log: "🐌 Descarga lenta detectada (5.2 KB/s) - Check 1/3"
   - Si velocidad mejora → Resetear contador

4. BÚSQUEDA DE ALTERNATIVA (después de 3 checks):
   - Cancelar descarga actual
   - Buscar mismo archivo en otros usuarios
   - Excluir usuario lento
   - Iniciar descarga con nuevo usuario
   - Log: "⚠️ Descarga demasiado lenta, buscando alternativa"

  private async Task CheckSlowDownloadAndFindAlternative(DownloadTask task)
  {
      try
      {
          // Verificar si ha pasado suficiente tiempo desde el inicio
          var elapsed = (DateTime.Now - task.StartTime).TotalSeconds;
          if (elapsed < SLOW_DOWNLOAD_CHECK_SECONDS)
              return;
          
          // Verificar si es momento de hacer otra verificación
          var timeSinceLastCheck = (DateTime.Now - task.LastSpeedCheck).TotalSeconds;
          if (timeSinceLastCheck < SLOW_DOWNLOAD_CHECK_SECONDS)
              return;
          
          // Calcular velocidad actual en KB/s
          long bytesSinceLastCheck = task.BytesDownloaded - task.LastBytesDownloaded;
          double speedKBps = (bytesSinceLastCheck / timeSinceLastCheck) / 1024.0;
          
          task.LastSpeedCheck = DateTime.Now;
          task.LastBytesDownloaded = task.BytesDownloaded;
          
          // Verificar si la velocidad es muy lenta
          if (speedKBps < MIN_SPEED_KBPS)
          {
              task.SlowDownloadChecks++;
              AutoLog($"🐌 Descarga lenta: {task.File.FileName} ({speedKBps:F2} KB/s) - Check {task.SlowDownloadChecks}/{SLOW_DOWNLOAD_THRESHOLD_CHECKS}");
              
              // Si se alcanza el umbral, buscar alternativa
              if (task.SlowDownloadChecks >= SLOW_DOWNLOAD_THRESHOLD_CHECKS)
              {
                  AutoLog($"⚠️ Descarga demasiado lenta, buscando alternativa: {task.File.FileName}");
                  UpdateDownloadUI(task, "🔍 Buscando alternativa (muy lento)...");
                  
                  // Cancelar descarga actual
                  task.CancellationToken?.Cancel();
                  
                  // Buscar proveedor alternativo
                  await TryFindAlternativeProvider(task);
              }
          }
          else
          {
              // Resetear contador si la velocidad mejora
              if (task.SlowDownloadChecks > 0)
              {
                  AutoLog($"✅ Velocidad mejorada: {task.File.FileName} ({speedKBps:F2} KB/s)");
                  task.SlowDownloadChecks = 0;
              }
          }
      }
      catch (Exception ex)
      {
          AutoLog($"⚠️ Error verificando velocidad: {ex.Message}");
      }
  }
