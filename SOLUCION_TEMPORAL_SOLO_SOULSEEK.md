# Solución Temporal: Usar solo Soulseek

Mientras configuras aMule correctamente, puedes usar SlskDown solo con Soulseek:

## Pasos:

1. **Abre SlskDown**

2. **Ve a Configuración**

3. **En la sección de Redes P2P**, cambia a:
   - **Red preferida**: Soulseek (o "Solo Soulseek")
   - Esto deshabilitará eMule temporalmente

4. **Guarda la configuración**

5. **Reinicia SlskDown**

Ahora las búsquedas funcionarán solo con Soulseek (que ya está funcionando correctamente).

## Cuando quieras volver a usar eMule:

1. Configura la contraseña de aMule Web correctamente
2. Actualiza la contraseña en SlskDown
3. Cambia la red preferida a "Ambas" o "eMule"

---

**Ventaja**: Puedes seguir usando SlskDown mientras resuelves el problema de aMule.
