## 🎯 Configuración Futura de eMule

Cuando instales eMule oficial:

### Paso 1: Instalar eMule
1. Descargar de https://www.emule-project.net/
2. Instalar normalmente
3. Ejecutar eMule

### Paso 2: Configurar External Connections
1. En eMule: **Opciones** → **Conexiones Remotas**
2. Configurar:
   ```
   ✅ Activar External Connections
   Puerto: 4712
   Contraseña: [tu_contraseña]
   ```
3. Aplicar y reiniciar eMule

### Paso 3: Conectar SlskDown
1. Abrir SlskDown
2. Ir a **Configuración**
3. Activar **"Habilitar eMule/ed2k"**
4. Ingresar contraseña EC
5. Reiniciar SlskDown
6. ¡Listo! 🚀
